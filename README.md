🛡️ **CreditAI: Intelligent Credit Scoring System**

Интеллектуальная система оценки кредитоспособности заемщиков на основе градиентного бустинга. Проект разработан как комплексное решение для автоматизации кредитных решений с акцентом на интерпретируемость и динамическое ценообразование.

---

## 📖 Обзор проекта

Система анализирует данные потенциального заемщика и предсказывает вероятность дефолта. В отличие от традиционных систем, данное решение объединяет мощь машинного обучения с прозрачностью бизнес-правил.

### Ключевые компоненты:
* **XGBoost Classifier**: Модель, обученная на данных Home Credit, достигшая **ROC AUC ~0.77**.
* **Risk-Based Pricing (RBP)**: Математический движок, который рассчитывает индивидуальную процентную ставку на основе вероятности дефолта. Клиенты с низким риском получают базовую ставку (от 14%), а для рисковых сегментов применяется риск-премия.
* **Rule-based Explanation Layer**: Слой интерпретации, который переводит сложные веса модели в понятные человеку причины (например, «Высокая долговая нагрузка» или «Низкий рейтинг внешних бюро»).
* **Explainable AI (XAI)**: Визуализация вклада каждого фактора в итоговый результат с помощью **SHAP Waterfall plots**.

## 📈 Логика принятия решений

### Risk-Based Pricing (RBP)
Система уходит от бинарного «Да/Нет». Для каждого одобренного кредита рассчитывается ставка по формуле:
> **Final Rate = Base Rate (14%) + Risk Premium**
Риск-премия растет нелинейно в зависимости от вероятности невозврата, что позволяет банку страховать риски, не отказывая клиентам.

### Rule-based Explainer
Для обеспечения детерминированности и прозрачности используется слой правил, который анализирует:
1.  **EXT_SOURCE_MEAN**: Средний балл из внешних кредитных бюро.
2.  **PAYMENT_RATE**: Отношение аннуитета к сумме кредита.
3.  **ANNUITY_INCOME_PERC**: Доля ежемесячного платежа в доходе.
4.  **DAYS_EMPLOYED_PERC**: Стаж работы относительно возраста.
